# Summary

Did this because I didn't find any on Thingiverse that actually held the spool with sufficient stability to perform solder in the lab. 
It looks flimsy, but is actually pretty sturdy. After you put the shaft, the bends will hold it in place.
Use support while printing, the ugly bits will be facing the board.

So far has been super useful =)
